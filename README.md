DESCRIPCION BREVE DEL PROYECTO:

Este proyecto consiste en el desarrollo de tres interfaces web para un sistema de gestión de solicitudes de crédito llamado "CreditSmart". La implementación utiliza HTML5 semántico, CSS3 profesional y diseño responsive para garantizar una experiencia de usuario óptima en dispositivos móviles, tablets y desktop.


ESTRUCTURA DE LOS ARCHIVOS:

CreditSmart/
-index.html              # Página principal - Catálogo de Créditos
-simulador.html          # Página de simulación - Búsqueda de Créditos
-solicitar.html          # Página de formulario - Solicitar Crédito
-styles.css              # Archivo principal de estilos CSS
-README.md               # Documentación del proyecto


INSTRUCCIONES PARA EJECUTAR EL PROYECTO

USANDO EL SERVIDOR LOCAL
Instalar una extensión de servidor local como "Live Server" en VS Code
Abrir la carpeta del proyecto en VS Code
Hacer clic derecho en index.html y seleccionar "Open with Live Server"
El proyecto se abrirá en http://localhost:5500 o puerto alguno parecido
